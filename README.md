# vpn_lists_JP

日本のIPアドレスを使えるVPNのリストです。

大手のVPNサービスはだいたい調査したつもりです。



表の見方ですが、ASNとVPNサーバーの場所が一致しているサービスは同じ行に書きました。

なので、ASNが同じでもサーバーの場所が違うサービスは違う行に書いています。

VPNサーバーの場所の見方は、かっこの外がVPNサービスが公称しているサーバーの場所で、かっこの中が実際のサーバーの場所です。

かっこがない場合は公称しているサーバーの場所と実際のサーバーの場所が一致しているサービスです。



AdGuard VPNの韓国サーバーに使われているIPアドレスですが、実際の場所は東京の可能性があるのでこの表に書きました。





実際のサーバーの場所やASNは"ipinfo.io"を使用して調査しました。
